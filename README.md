# ライフマネープラン

家計簿ではなく「人生設計・人生シミュレーター」。
お金を管理するのではなく、人生で何がいつ起きて、そのときお金がどうなるかを一枚で見るためのツール。

## コンセプト

- お金は「点」ではなく「流れ」として見る
- 過去・現在・未来を同じ画面で見る
- 未来予測は「当てる」のではなく「比較する」(年利・退職年齢などのシナリオを変えて比較する)
- 節約ではなく「人生の選択」を支援する
- 最終的に「あといくら使っていいか」が分かることを目指す

## 構成(ページ)

| ファイル | 内容 | 状態 |
|---|---|---|
| `index.html` | 投資シミュレーション。現在の資産・月々のキャッシュフロー(ボーナス含む)・想定年利から将来の資産推移をグラフ化 | 完成 |
| `timeline.html` | 人生キャッシュフロー。年×収入・支出・資産の横スクロールグリッド。家族の生年から年齢を自動計算し、収入・支出はフォワードフィル(入力した年以降ずっとその額。途中の年を変えればそこから先が切り替わる)、NISA・個別株などの投資資産を年率%で複利計算して総資産に統合。年次↔月次プランナーの相互反映にも対応 | Phase 1-3 完成 |
| `kakeibo.html` | 家計簿。日々の支出記録+固定費+月合計を月単位で記録。分類(カテゴリ)は人生キャッシュフローと共通で、大項目ごとに整理・追加・削除できる | 完成 |

## 技術メモ

- 単一HTMLファイル(HTML+CSS+JS)、ビルド不要
- グラフ表示: Chart.js(CDN読み込み、ネット接続が必要)
- データ保存: ブラウザのlocalStorage(端末・ブラウザごとに保存)
- ローカル確認: `python -m http.server 8767`(`.claude/launch.json` に設定済み、`timeline.html` も同じサーバーで開ける)
