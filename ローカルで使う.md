# 自分のPCだけで使う方法（ネット不要・データは外に出ません）

奥さんのPCなど、**インターネットにつながずに、その端末だけ**で使うための手順です。

## 安心ポイント
- 入力した金額は、その**PCのブラウザの中だけ**に保存されます。ネットにもGitHubにも送信されません。
- グラフ表示の Chart.js も**同梱済み**なので、インターネット接続は不要です（外部サイトへ一切アクセスしません）。

## 手順
1. **このフォルダ一式**を、使いたいPCにコピーします（USBメモリやクラウド経由など）。
   - 次のファイルが**すべて同じフォルダ**に入っている必要があります：
     - `index.html`（投資シミュレーション）
     - `timeline.html`（月次・年次キャッシュフロー）
     - `kakeibo.html`（家計簿）
     - `chart.umd.min.js`（グラフ部品。これが無いとグラフが出ません）
2. `kakeibo.html`（または `timeline.html`）を**ダブルクリック**して、Google Chrome で開きます。
   - 左のメニュー（サイドバー）から 家計簿 / 月次 / 年次 / 投資 を行き来できます。
3. 最初に**合言葉**を聞かれたら入力してください（一度入れたらその端末では記憶されます）。

## 注意
- ファイルは**バラバラにせず、必ず同じフォルダにまとめて**置いてください。
- ブラウザは **Google Chrome** を推奨します。
- データはブラウザごと・端末ごとに保存されます。別のPCやブラウザには引き継がれません（それぞれ独立）。
- 中身を更新したいときは、新しいフォルダ一式で上書きコピーすればOKです（入力データはブラウザ側に残ります）。

## データのバックアップ（任意）
入力データはブラウザ内（localStorage）にあります。今は端末内のみの保存です。
別端末への引っ越しやバックアップが必要になったら、書き出し／読み込み機能を追加できます（要相談）。
